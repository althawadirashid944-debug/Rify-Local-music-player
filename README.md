# Rify

A simple music downloader and player built with Python, GTK4, yt-dlp and mpv.

## Download


### Linux

Install dependencies (this is one way to do it) 

```bash
sudo pacman -S python-gobject gtk4 mpv 
```bash
CLONE THE REPOSITORY
```bash
git clone https://github.com/althawadirashid944-debug/Rify-Local-music-player.git 
cd Rify-Local-music-player
```
Run the main.py file 
You may also asign a shortcut to the main.py file for easier access.
(Note: You may need to install yt-dlp if you don't have it already. You can do this by running `pip install yt-dlp`.)

### 
Second way to install:
Go into the packaging folder there will be 2 folders one for linux and one for windows.
Linux users open the linux folder and run the .sh file (auto downloads dependencies)
